Pacote com compêndios exclusivos do sistema de RPG Caves & Hexes, desenvolvido para ser executado no FoundryVTT através do sistema Old School Essentials (OSE).
Sistema originalmente desenvolvido por:
- Carlos “Malvadeza” Silva;
- Francisco Siqueira;
- João Burlamaqui;
- Rafael Balbi;
- Rafael Massuia.

Para mais informações sobre o sistema Caves & Hexes e acesso ao SRD gratúito do sistema, acesse: https://www.cavesandhexes.com
